Usage
 Coding(insert mode)
   ``         -> autocomplete function name and see scrach
   Ctrl+p     -> autocomplete function name
   Ctrl+o     -> comment block
   Ctrl+v     -> block selection
   Ctrl+c     -> close bracket
   F2         -> Toggle paste mode
   Tab        -> for fun arr array ... see snippets/xxx.snippets

 PHP
   Ctrl+l     -> verify syntax error in php-cli
 
 Tab
   Ctrl+t     -> new tab
   `t | `]    -> next tab
   `[         -> previous tab
 
 File
   F1         -> NerdTree file directory
   `f         -> Fuzzysearch
   `F         -> Fuzzysearch recursive
